JoinLeaveSounds
-------------------------------------------------
Simple UdonSharp script to play audio clips when players join or leave.
-------------------------------------------------
Setup:

1) Place JoinLeaveSounds.cs on any active GameObject.
2) Assign an AudioSource component.
3) In the Inspector, set the four audio clips:
	- selfJoinClip
	- otherJoinClip
	- selfLeaveClip
	- otherLeaveClip
	- (Optional) Enable Randomize Pitch and adjust Pitch Range.
	- (Optional) Toggle Debug Log for console messages.

That’s it—no additional steps required.
